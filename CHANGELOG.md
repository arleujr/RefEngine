# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-07-18

### Added

- Local PDF, BibTeX, and RIS processing.
- Native PDF text extraction and optional local OCR.
- Structured-source consolidation to avoid duplicate references.
- UFV 2025 catalog with 43 renderable schemas.
- React and TypeScript review interface.
- Field-specific conflicts, missing fields, and source alternatives.
- Explicit save and approval workflow.
- Exclusion without mandatory-field validation.
- Alphabetically ordered DOCX and TXT publication.
- FastAPI local API, SQLite persistence, immutable run snapshots, and OpenAPI types.
- Locked Python and Node.js dependencies.
- Backend and frontend continuous integration checks.
