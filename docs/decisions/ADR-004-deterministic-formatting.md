# ADR-004: Deterministic reference formatting

## Status

Accepted.

## Decision

Reference output is generated by versioned deterministic rules. Language models cannot directly author the final formatted reference.

## Rationale

Formatting must be reproducible, auditable, and testable. Generative output can vary and may silently invent missing values.

## Consequences

- rules require explicit implementation;
- model-assisted extraction remains optional;
- formatting regressions can be covered by tests.
